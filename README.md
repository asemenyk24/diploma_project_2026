**Тема ВКР:** Выявление и анализ визуальных паттернов в архитектуре для определения периода постройки и архитектурного стиля методами глубинного обучения

### План выполнения работы  

### Этап 1: Обзор источников и существующих решений (23.01.2026) ✅
* **Анализ доступных источников:** WikiArt, Flickr API, Cities and Buildings Database и др.  
* **Изучение существующих решений:** классические CNN (LeNet-5, AlexNet), современные гибридные архитектуры (HistoNet) и генеративные модели.  
* **Результат:** сформирован содержательный литературный обзор (21 источник).  

### ЭТАП 2: Сбор данных, формирование датасетов, разведочный анализ данных (07.03.2026) ✅
* **Сбор данных:** автоматическое извлечение атрибутов зданий через OpenFACADES, ручная корректировка в CVAT.  
* **Формирование датасета:** 16113 изображение, 11 информативных признаков, 10 основных типов зданий по назначению.  
* **EDA:** визуализация распределений по классам, материалам, этажности и годам постройки. Выявлен критический дисбаланс классов и аномалии разметки.
* **Результат:** раздел ВКР "Исследовательский анализ данных" + ноутбук "eda.ipynb".

### ЭТАП 3: Согласование метрик, разработка baseline-решения  (08.04.2026 - 03.05.2026) ✅
* **Метрики:** Accuracy, Precision, Recall, F1-score.  
* **Baseline-решение:** ResNet-50 с предобученными весами ImageNet.
  * Взвешенная функция потерь и Weighted Random Sampler для борьбы с дисбалансом.
  * Аугментации: RandomCrop, RandomHorizontalFlip, ColorJitter.
  * Регуляризация: Dropout (0.3), AdamW, ReduceLROnPlateau, ранняя остановка.
* **Результаты:** точность на тестовой выборке — **72.3%**.
  * Топ класс: apartments - F1 = 0.794.
  * Худший класс: education - F1 = 0.527.
  * Самый редкий класс: garage (51 пример) - F1 = 0.733.
* **Результат:** раздел ВКР "Baseline-решение" + ноутбук "baseline.ipynb".

### ЭТАП 4: Содержательная часть ВКР  (04.05.2026 - 13.05.2026) ✅
* **Обучение многозадачной модели:** ConvNeXt-Base, одновременно предсказывающая 8 атрибутов зданий.
  * Точность классификации типов зданий: **72.9%** (+1.1% к baseline).
  * Классификация материалов отделки: **83.2%**.
  * Классификация материалов конструкции: **86.1%**.
  * Предсказание этажности: MAE **1.24** этажа (R^2 = **0.868**).  
* **Интерпретация результатов:** Grad-CAM для визуализации областей внимания модели.
  * Подтверждено: модель опирается на архитектурно значимые элементы фасада.
  * Выявлены причины ошибок: визуальная схожесть классов "education" и "public", смещение фокуса внимания на посторонние объекты.
* **Результат:** раздел ВКР "Предсказание множественных признаков" + ноутбук "final_model.ipynb".
